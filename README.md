# Experiment 1.2.1

## Live Character Counter

### Aim
To implement a live character counter that updates as users type in a text area.

### Objectives
- Create a responsive text input component  
- Implement real-time character counting  
- Provide visual feedback for character limits  

### Tools Used
- HTML  
- CSS  
- JavaScript  

### Output
A text area that dynamically displays the number of characters typed by the user.
